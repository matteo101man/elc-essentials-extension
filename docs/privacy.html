Privacy Policy — ELC Essentials

Last updated: April 21, 2026

ELC Essentials is a browser extension that enhances the University of Georgia D2L Brightspace site (uga.view.usg.edu) and related Athena pages (athena-prod.uga.edu) with student tools such as practice quizzes, a course schedule view, and a grade calculator. This policy describes how the extension handles information.

Who we are

ELC Essentials is developed independently and is not affiliated with D2L, the University System of Georgia, or the University of Georgia. The developer is Matteo Garza (as listed in the extension). For privacy questions, open an issue on the GitHub repository where this extension’s source is published (use the Issues tab on that repo).

Data stored on your device

The extension keeps data locally in your browser to provide its features, including:

  • Enable/disable state — whether the extension is turned on (via chrome.storage.local).
  • Practice test data — cached quiz content and related settings you use with the practice test feature.
  • Course schedule — schedule information you import or build, synchronized in storage so it is available across D2L and Athena tabs.
  • Cached D2L course lists — short-lived cache to speed up the schedule view.
  • Session-style keys — small amounts of data in page storage (namespaced) so features work correctly on the site.

This data stays on your device unless you use optional features that send data elsewhere (see below). Uninstalling the extension or clearing extension storage removes this local data.

Optional third-party services

OpenAI: If you choose optional AI-related features and enter your own OpenAI API key, requests are sent from the extension to api.openai.com only when you use those features. Those requests are subject to OpenAI’s privacy policy. The developer does not receive your API key or the content of those requests on any server operated by the developer.

Remote JSON (GitHub): If you provide a URL to load practice exam JSON from raw.githubusercontent.com or gist.githubusercontent.com, the extension fetches that file when you request it. Only URLs you supply are used.

Local development: Optional access to http://127.0.0.1 and http://localhost is for local tools you may configure; no data is sent to the developer by default.

What we do not do

  • We do not sell your personal information.
  • We do not use the extension to show unrelated advertising.
  • We do not run analytics or tracking scripts in the extension for advertising purposes.
  • Executable code is shipped inside the extension package; the extension does not download and execute remote JavaScript as extension code.

Permissions (summary)

  • Storage — save settings and cached data locally.
  • Tabs — switch or close tabs when importing a schedule from Athena back to D2L, as described in the extension.
  • Host access — interact only with the sites and APIs listed in the extension manifest (D2L, Athena, optional OpenAI, optional GitHub raw/gist, optional localhost).

Children

The extension is intended for students using university systems. It is not directed at children under 13 in a standalone product sense; if you are a parent and have concerns, contact us via the GitHub repository.

Changes

We may update this policy when the extension or legal requirements change. The “Last updated” date at the top will change when we do. Continued use of the extension after updates means you accept the revised policy.
